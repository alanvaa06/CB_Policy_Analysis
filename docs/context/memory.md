# constraint: live FOMC signals limited to PUBLIC real-time corpus (statements May-1999+, minutes ~3wk lag, press-conf transcripts 2011+, speeches); meeting TRANSCRIPTS + Tealbook/Bluebook alternative statements carry ~5yr release lag → historical-research-only, not live signal.
# decision (pending): backbone model candidate = FOMC-RoBERTa "Trillion Dollar Words" (gtfintechlab, ACL 2023) — but CC BY-NC 4.0 = non-commercial only; blocks commercial deployment without relicense.
# finding: all academic "predictive" results to date are in-sample/correlational (Lucca-Trebbi ~30bp, IMF ~20bp); no common-corpus OUT-OF-SAMPLE FOMC head-to-head benchmark exists → that gap is the main innovation white-space.
# decision: Phase 0 eval harness BUILT + VALIDATED (offline synthetic-recovery + null-rejection; live confounded signal → no false positive). Apparatus is trustworthy; build on it for Phase 1.
# finding: combined-doc-type TDW stance (minutes+pressers+speeches averaged per date) has NO usable OOS forward power on DGS2 (negative OOS R², sub-0.5 hit) — confounded by mixed doc timing/role. Phase 1 must use a CLEAN single-doc-type, market-timed stance (statements or pressers), NOT a doc-mixed average.
# finding: per-doc-type diagnostic — minutes carry NO OOS signal (stale ~3wk-lagged release); pressers' only strong effect (EFFR event-study t=4.84) is MECHANICAL (text co-moves with same-day rate decision), not predictive; DGS2 shows no robust effect for any doc type. Phase 1 must CONTROL FOR the policy surprise to isolate text's marginal effect, target DGS2/OIS (not EFFR), drop minutes, add standalone FOMC statements.
# lesson-source: offline test fixtures used tz-aware market index; real FRED is tz-naive → integration bug only the live run caught. Prefer fixtures that mirror real data shape.
# decision: Phase 1 harness BUILT (branch phase-1-marginal-effect, 59 tests) — isolates statement TEXT's marginal effect via nested OOS (ΔR² of surprise+stance vs surprise-only; control = Bauer-Swanson ORTHOGONALIZED surprise); EFFR dropped, targets DGS2/DGS1; FOMC-RoBERTa stance = per-sentence-mean on standalone statements 1999+ (transformers lazy + injected classifier → tests fake-only, heavy deps in optional [infer] extra). CC BY-NC still blocks commercial deploy.
# VERDICT: Phase 1 NO-GO (2026-06-19, docs/results/2026-06-19-phase1-verdict.md). FOMC statement stance (FOMC-RoBERTa per-sentence-mean) adds NO OOS value beyond BS orthogonalized surprise: ΔR²<0 at ALL 6 cells (DGS2,DGS1 × h1,5,22), n=177, 197 statements 1999-2023. Valid publishable null. Canonical-confirmed 2026-06-29: gated gtfintechlab/FOMC-RoBERTa reproduces the mirror tim9510019 result IDENTICALLY (all 6 cells, 4 decimals) → byte-equivalent, provenance caveat CLOSED. Remaining caveats: per-sentence-mean is crude; BS surprise intraday-calibrated vs daily DGS.
# VERDICT: Phase 2a lexicon baseline NO-GO (2026-06-29, docs/results/2026-06-29-lexicon-baseline-verdict.md). Transparent policy-stance word-count tone (hawk tighten/restrictive/firming, dove accommodative/easing/stimulus) shows apparent ΔR²>0 at h=1 (DGS1 +0.055) where RoBERTa was flat-null — but it is a REGIME CONFOUND: stance ≈ degenerate 3-level easing-vs-hiking-era indicator (corr +0.65 with rate level), the h=1 gain is absorbed by a rate-level control and sign-flips when pre-2008 is dropped, with a backwards coefficient. NOT a robust signal. Value = interpretability: being transparent let us DIAGNOSE the confound (impossible with the RoBERTa black box). [[decision-lexicon-confound-control]]
# lesson: a positive OOS ΔR² on a slow-moving/near-degenerate text feature must be hardened against REGIME confound (control for cycle/rate-level + era jackknife) before any GO — an apparent edge can be era-vocabulary leaking past a single-number surprise control.
